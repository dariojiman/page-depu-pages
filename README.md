# page-depu-pages

## 项目简介

这是一个用于归档和发布独立 HTML 页面的仓库。本仓库不针对任何特定域名或网站，仅作为通用页面存储与展示的空间。

## 目录说明

```
page-depu-pages/
├── pages/          # 存放归档的 HTML 页面及相关资源
├── assets/         # 公共资源文件（如 CSS、JS、图片等）
└── README.md       # 本文件
```

- `pages/`：每个子目录对应一个独立页面，包含其 HTML、样式、脚本等文件。
- `assets/`：存放可被多个页面引用的公共资源，方便复用和维护。

## 页面归档说明

- 所有页面均为独立的 HTML 文件，不依赖外部特定环境。
- 页面内容可能涉及临时存档、测试用例、示例展示等，不保证长期有效。
- 归档页面可能包含静态资源，建议使用相对路径引用，确保本地或部署后均可正常访问。

## 维护说明

- 欢迎提交 Issue 或 Pull Request 来添加、更新或移除页面。
- 添加页面时，请在 `pages/` 下创建独立子目录，并确保所有资源文件完整。
- 公共资源如有更新，请同步修改引用路径或版本号。
- 本仓库不承诺主动更新页面内容，请贡献者自行保证所提交内容的合规性与准确性。

## 其他

- 本仓库使用 GitHub Pages 进行部署，可通过仓库设置开启 Pages 功能。
- 部署后，每个页面可通过 `https://<用户名>.github.io/page-depu-pages/pages/<页面目录>/` 访问。
- 如有疑问或建议，欢迎通过 Issue 反馈。
